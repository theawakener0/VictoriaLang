# Victoria Language Examples

This directory contains example scripts demonstrating various features of the Victoria programming language.

## Basic Examples

- `hello.vc`: A simple "Hello, World!" program.
- `fib.vc`: Calculates Fibonacci numbers using recursion.
- `math_test.vc`: Demonstrates mathematical operations.
- `try_catch.vc`: Shows error handling capabilities.

## Data Structures

- `structs.vc`: Demonstrates how to define and use structs and methods.
- `include_test.vc`: Shows how to include other files.

## Networking

- `server.vc`: A simple HTTP server implementation.
- `html_server.vc`: An HTTP server serving HTML content.
- `tcp_server.vc`: A TCP server example.
- `tcp_client.vc`: A TCP client example.
- `http_client_test.vc`: Demonstrates making HTTP requests.

## Testing

- `full_test.vc`: A comprehensive test suite for language features.
- `stdlib_test.vc`: Tests for the standard library.
- `modules_test.vc`: Tests for module loading.

## How to Run

To run any of these examples, use the `victoria` interpreter from the root directory:

```bash
./victoria examples/hello.vc
```
